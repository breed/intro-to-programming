# Study Guides

- there is a study guide for each textbook chapter
- it should ask open ended questions about about the key concepts of the chapter
- the answer to the question should be covered in the chapter
- every important topic should have at least one question
- a topic or concept may need multiple questions or experiments to cover various aspects of the topic
- when appropriate, suggest experiments to run or programs to write
- the questions should be ordered according to the order their answers appear in the text
- the study guide should only have questions and suggestions for experiments, it should not have notes or answers

## File names

study guide markdown files have the name `sg-chXX.md` where XX is the chapter number
